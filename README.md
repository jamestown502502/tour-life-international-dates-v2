# Tour Life: International Dates

> *Live loud. Love wild. Headline the world.*

A browser-based 2D rhythm RPG.

## Controls

City Hub: WASD / Arrow keys to move, E to interact

Rhythm: D F J K for lanes 1-4

Dialogue: UP/DOWN navigate, SPACE/ENTER confirm

## Stack
Phaser.js 3.80 + Vite 5 + Vercel

## Story
Six cities. Three bandmates. One shot at global fame.
